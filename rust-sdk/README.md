# Jupiter RFQ v2 Rust SDK

A Rust SDK for interacting with Jupiter's RFQ (Request for Quote) v2.

## Examples

Check out the `examples/` directory for complete examples:

- [`production_streaming.rs`](examples/production_streaming.rs) - Production-ready streaming with price feeds and volume-based pricing

Run an example:

```bash
cargo run --example production_streaming
```

## Requirements

- Rust 1.70 or later
- Tokio runtime for async operations
